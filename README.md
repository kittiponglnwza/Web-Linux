# 🐧 Web Linux Quest

เว็บไซต์เรียน Linux แบบ Gamification ที่ช่วยให้ผู้เรียนฝึกคำสั่ง Linux ผ่านบทเรียน ภารกิจ และ Challenge แบบโต้ตอบ พร้อมระบบ XP และ Rank เพื่อเพิ่มความสนุกในการเรียนรู้

## 🚀 Features

* 📚 บทเรียน Linux 10 Chapters
* 🎯 Quest & Challenge
* ⭐ XP & Rank System
* 📝 Quiz และแบบทดสอบ
* 💻 ฝึกใช้คำสั่ง Linux จริง

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* JSON
* Vercel
